# react-native-hide-under-status-bar

Hide a text under status bar

## Installation

```sh
npm install react-native-hide-under-status-bar
```

## Usage

```js
import HideUnderStatusBar from "react-native-hide-under-status-bar";

// ...

const result = await HideUnderStatusBar.multiply(3, 7);
```
